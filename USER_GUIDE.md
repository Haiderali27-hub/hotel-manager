## üîß First-Time Setup

Before you start managing guests, you need to configure your hotel:

### 1. Set Up Rooms
1. Click on the **Settings** menu (gear icon)
2. Select **"Manage Menu/Rooms"**
3. Go to the **"Rooms"** tab
4. Click **"Add Room"** for each room you want to add
5. Enter:
   - **Room Number** (e.g., 101, 102, etc.)
   - **Room Type** (Standard, Deluxe, Suite, or create custom)
   - **Daily Rate** (price per night)
6. Click **"Save"** for each room

### 2. Create Restaurant Menu
1. In the **"Manage Menu/Rooms"** section
2. Go to the **"Menu"** tab
3. Click **"Add Menu Item"**
4. Fill in:
   - **Item Name** (e.g., "Chicken Biryani")
   - **Category** (Breakfast, Main Course, Appetizers, Desserts, Beverages)
   - **Price** (in Pakistani Rupees)
   - **Description** (optional)
5. Repeat for all menu items

### 3. Configure System Settings
1. Set **tax rates** if applicable
2. Configure **currency format** (default: Pakistani Rupees)
3. Set up **backup preferences**

## üë• Managing Guests

### Adding a New Guest (Check-in)

1. **Click "Add Guest"** from the main dashboard
2. **Fill in guest information**:
   - **Name**: Guest's full name
   - **Phone**: Contact number
   - **Email**: Email address (optional)

3. **Select accommodation details**:

## üîß Troubleshooting

### Common Login Issues

**Problem**: Cannot log in with credentials

1. Verify username: `yasinheaven`
2. Check password: `YHSHotel@2025!`
3. Use password recovery if needed

**Problem**: Application won't start
**Solution**:
1. Check if application is already running
2. Restart computer
3. Reinstall application if necessary

### Guest Management Issues

**Problem**: Cannot add new guest
**Solution**:
1. Ensure all required fields are filled
2. Check if room is available
3. Verify dates are in correct format
4. Restart application if error persists

**Problem**: Guest doesn't appear in active list
**Solution**:
1. Confirm check-in was completed successfully
2. Check if guest was marked as checked out
3. Search in guest history
4. Verify database connection

### Food Order Problems

**Problem**: Menu items missing
**Solution**:
1. Check if items are marked as available
2. Verify menu items were saved properly
3. Refresh the menu list
4. Re-add missing items if necessary

**Problem**: Order total incorrect
**Solution**:
1. Verify item prices in menu settings
2. Check if tax is being applied correctly
3. Confirm quantity calculations
4. Review order items list

### Report Generation Issues

**Problem**: Reports showing incorrect data
**Solution**:
1. Verify date range selection
2. Check if all transactions were recorded
3. Ensure expenses are properly categorized
4. Refresh report data

**Problem**: Cannot export reports
**Solution**:
1. Check available disk space
2. Verify export folder permissions
3. Try different export format
4. Restart application

### Performance Issues

**Problem**: Application running slowly
**Solution**:
1. Close other applications to free memory
2. Restart the hotel management system
3. Check available disk space
4. Consider computer upgrade if persistent

**Problem**: Database errors
**Solution**:
1. Backup current data immediately
2. Restart application
3. Contact technical support
4. Restore from backup if necessary
